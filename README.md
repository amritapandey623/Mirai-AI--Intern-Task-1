# Identity Echo Interface

A Streamlit web application created for the MirAI School of Technology Virtual Summer Internship 2026.

## Features

- Takes user's name
- Takes user's message
- Validates empty inputs
- Displays personalized success message
- Estimates AI token consumption

## Installation

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```

The application will open in your browser at:

```
http://localhost:8501
```
